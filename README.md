# Proiect-BD-farmacie

## 1. Descrierea modelului real, a utilitatii acestuia si regulilor de functionare

O apliceatie pentru gestionarea personalului cat si al stockului si vanzariilor intr-o farmacie, ofera utilizatorilor (farmacistilor) posibilitatea sa gestioneze toate informatiile necesare pentru buna functionare a farmaciei.
